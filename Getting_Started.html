First off, HTML is not a programming language, it is a hypertext markup language. The most basic format of it follows this principle: It defines the langauge (html), the page title, the headings, the body and the text. You can try it yourself in the box below!

To begin making your own webpage, you will need an IDE (integrated development environment) these softwares are specifically designed to write code in. My recommendation is the Atom software, as it is free to download on both mac and windows as well as linux, and it gives tag suggestions making the coding process much faster

Download atom HERE

If you plan on coding in python, SublimeText is a much better option. It is free to download HERE

The beauty of an HTML document is that it opens directly in a browser, allowing you to see the progress you have made. Everytime you make a change, you can save the document, and refresh it in your browser to show the new changes. This makes testing the code a whole lot faster than most other languages
